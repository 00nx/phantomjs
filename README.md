PhantomJS is a **headless** WebKit ‚Äî a full browser with no GUI ‚Äî scriptable using JavaScript.  
The last stable release is [2.1](http://phantomjs.org/release-2.1.html).

---

## üìå Use Cases

- **Headless web testing** ‚Äî Run fast, automated tests without needing a visible browser.
- **Page automation** ‚Äî [Access and manipulate](http://phantomjs.org/page-automation.html) pages with the DOM API or libraries like jQuery.
- **Screen capture** ‚Äî Programmatically [capture web content](http://phantomjs.org/screen-capture.html) ‚Äî HTML, CSS, SVG, Canvas.
- **Network monitoring** ‚Äî Automate performance analysis, monitor [page loading](http://phantomjs.org/network-monitoring.html), and export as HAR.

## ‚ú® Features

- **Cross-platform** ‚Äî Windows, macOS, Linux, and other Unices.
- **Native web standards** ‚Äî DOM, CSS, JavaScript, Canvas, SVG ‚Äî no emulation.
- **Pure headless on Linux** ‚Äî No X11 needed. Ideal for CI/CD. Runs on EC2, Heroku, Iron.io.
- **Quick start** ‚Äî [Download](http://phantomjs.org/download.html), unpack, run.
